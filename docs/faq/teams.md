# Teams 会議に参加できないとき

Microsoft Teams（社内で使う通話・チャットツール）で会議に参加できない／音が聞こえないときの手順です。できるだけシンプルにまとめているので、上から順に試してください。

## よくあるパターン

- 会議リンクを押してもブラウザが開くだけでアプリが起動しない。
- 参加できてもマイクやスピーカーが無効になっている。
- 「組織のポリシーでブロックされています」という表示が出る。

## 参加できないときのチェック

1. **アプリの更新**
   - 画面右上の自分のアイコン → `更新を確認` をクリック。最新版にしないと新しい機能（例: Together モード）が使えない場合があります。
2. **キャッシュをクリア**（キャッシュ = 一時的に保存されたデータ）
   - Windows: `設定 > アプリ > Teams > 詳細オプション > リセット` を選択。
3. **リンクの開き方を変更**
   - Outlook からリンクを右クリックし、「コピー」を選んでブラウザに直接貼り付けると解決するケースがあります。
4. **別のアカウントでログインしていないか確認**
   - 会社アカウント以外（例: プライベート Microsoft アカウント）でログインしていると組織制限に引っかかることがあります。`サインアウト` → `再ログイン` でやり直してください。

## 音声トラブルの対処

1. Teams 画面右上の **「…」> 設定 > デバイス** を開き、入力（マイク）・出力（スピーカー）の欄を手動で選び直します。
2. OS 側のサウンド設定でミュートになっていないか確認します。Windows の場合はタスクバー右下のスピーカーアイコンをクリックします。
3. Bluetooth イヤホンを使っている場合は、一度切断→再接続してから、Teams 上でもデバイスを再選択します。
4. どうしても改善しない場合は、電話参加用のダイヤルイン番号を使って音声だけ参加してください。

## 連絡先

- エラーコード（例: `CAA5004B`）が表示されたら控えてください。
- `![Teams エラー画面例](../../assets/images/teams-join.svg "Teams 参加画面例")` を参考に、似た画面が出ているか確認しながらスクリーンショットを添付してください。
- 緊急の会議で困ったら Slack `#helpdesk-sos` に投稿するか、内線 4242（ヘルプデスク直通）へ連絡してください。
