# プリンタで印刷できないときの基本手順

オフィスの共有プリンタから印刷できない場合に、現場で素早く確認するための手順です。難しい専門用語には補足を入れています。

## 症状別のヒント

- 紙が出てこない／途中で止まる
- ジョブ（印刷の予約）が 0% のまま進まない
- 印刷は完了した表示なのに白紙が出てくる

## すぐに試せること（3 分）

1. **プリンタの状態を確認**
   - 液晶に「Paper Jam（紙詰まり）」などが出ていないか見ます。英語の場合は意訳メモ（`Paper Jam = 紙詰まり`）を横に貼ってあります。
2. **紙トレイとトナー**
   - トレイに紙が入っているか、トナー／インク残量が極端に減っていないかを確認。足りない場合はヘルプデスクの備品棚から補充してください。
3. **再接続**
   - Windows の「デバイスとプリンター」で対象プリンタを右クリック → `トラブルシューティング` を実行。

## PC 側のチェック

1. **プリンタキューをクリア**
   - `Win + R` → `services.msc` → `Print Spooler` を右クリックし「再起動」。Spooler（スプーラー）は印刷待ち行列を管理するサービスです。
2. **ドライバーの再インストール**
   - メーカーサイトから最新ドライバーを入れ直します。手順は [assets/snippets/printer-driver.md](../../assets/snippets/printer-driver.md) にまとめています。
3. **別ユーザーでテスト**
   - 自分の PC だけの問題か確認するため、同僚の PC から同じプリンタに印刷してもらいます。

## それでも印刷できないとき

- エラーコードや本体の状態ランプをスマホで撮影し、Issue テンプレートの「添付情報」にアップロードしてください。`![プリンタ状態パネル](../../assets/images/printer-panel.svg "PAPER JAM の例")` を参考に、どのランプが点灯しているかメモすると伝わりやすいです。
- 紙詰まりが解消できない場合は、プリンタ上部のカバーを開けずにヘルプデスクへ連絡してください。無理に取り出すと感光ドラムを傷つける恐れがあります。
- 重要書類を急ぎで出したい場合は、別フロアのバックアッププリンタ `PRN-B2` を利用できます。
