# VPN 接続ができないときのチェックリスト

このページは、社内ネットワークへ入るための VPN（離れた場所から安全につなぐ仕組み）がうまく動かないときに使います。専門用語が出た場合はかんたんな説明を添えているので、順番に試してください。

## よくある症状

- 接続ボタンを押しても「認証に失敗しました」と表示される。
- いつもより接続完了まで時間が掛かり、そのままタイムアウト（制限時間超過）になる。
- 自宅 Wi-Fi ではつながらないが、スマホのテザリングではつながる。

## 事前チェック（1 分）

1. 社員証アプリで **ワンタイムパスワード**（30 秒ごとに変わる数字）を確認し、入力ミスがないかチェック。
2. ノート PC の時計が正しいか確認。ズレがあると認証サーバーが拒否する場合があります。
3. 会社から配布された VPN クライアント以外は終了させておきます。

## 解決ステップ

1. **アプリを再起動**
   - VPN アプリを完全終了→再起動します。Windows ならタスクバーの隠れたアイコンから終了してください。
2. **ネットワークを切り替え**
   - Wi-Fi を一度 OFF → ON。可能ならスマホのテザリングで試し、回線が原因か切り分けます。
3. **資格情報を更新**
   - `Ctrl + Alt + Delete` → 「パスワードの変更」を選び、社内パスワードを再入力。サーバー側と同期されます。
4. **VPN プロファイルを再読み込み**
   - VPN アプリ内の「設定」→「プロファイルの更新」をクリック。プロファイルとは接続先情報のことです。
5. **端末を再起動**
   - 5 分以上つながらない場合は PC 全体を再起動してください。

## まだ解決しない場合

- 画面右下に表示されるエラー番号（例: `-619`）をメモし、ヘルプデスクに共有してください。
- `![VPN エラー画面](../../assets/images/vpn-status.svg "VPN エラー画面サンプル")` のようなスクリーンショットを添付すると調査が早く進みます。
- 緊急で社内システムへ入りたい場合は、一次連絡先として Slack `#helpdesk-sos` チャンネルに投稿してください。
- Issue を登録するときは [assets/snippets/template.md](../../assets/snippets/template.md) をコピーして使うと、必要項目の漏れを防げます。
