# プリンタドライバー再インストール手順（社内標準）

1. メーカー公式サイト（例: [https://example.com/printer-driver](https://example.com/printer-driver)）を開き、機種名 `HDMC-4100` を検索します。
2. OS を選び、`Full Package`（フルパッケージ）をダウンロードします。サイズが大きいときは社内ファイルサーバー `\\fileserver\drivers` から入手しても構いません。
3. ダウンロードしたファイルを右クリックし「管理者として実行」。
4. 途中で「既存のドライバーを削除しますか？」と聞かれたら `はい` を選びます。
5. インストール完了後に PC を再起動し、`デバイスとプリンター` からテスト印刷します。

> memo: 管理者権限がない場合はヘルプデスクに申請してください。リモートで代行インストールします。
